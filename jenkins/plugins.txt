# Installation Instructions:
# 1. Log into Jenkins as admin
# 2. Navigate to Manage Jenkins > Manage Plugins > Available
# 3. For each plugin listed, search by name and install
# 4. Alternatively, use Jenkins CLI:
#    java -jar jenkins-cli.jar -s http://localhost:8080 install-plugin <plugin-name>
# 5. After installing all plugins, restart Jenkins:
#    Manage Jenkins > Restart Jenkins

# Note: Some plugins listed may be dependencies and installed automatically when you install others.
# Plugin versions are not pinned to ensure compatibility with latest Jenkins LTS.
